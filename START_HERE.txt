🚗 FIXACAR SKU PREDICTOR v3.0 - PORTABLE PYTHON EDITION
================================================================

📋 QUICK START GUIDE:

1️⃣ FIRST TIME SETUP:
   Double-click: 0_setup_portable_environment.bat
   (This installs Python packages - only needed once)

2️⃣ GET DATA:
   Double-click: 1_run_consolidado_downloader.bat
   (Downloads latest data from server)

3️⃣ PROCESS DATA:
   Double-click: 2_run_data_processor.bat
   (Converts data to optimized database)

4️⃣ TRAIN VIN MODELS:
   Double-click: 3_run_vin_trainer.bat
   (Trains VIN prediction models)

5️⃣ TRAIN SKU MODELS:
   Double-click: 4_run_sku_trainer.bat
   (Trains SKU prediction models)

6️⃣ RUN APPLICATION:
   Double-click: run_sku_predictor.bat
   (Main application for daily use)

📁 FOLDERS:
   • portable_app/    - Main application and data
   • portable_python/ - Python environment
   • Documentation/   - User guides and manuals
