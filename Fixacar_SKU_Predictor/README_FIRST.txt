Fixacar SKU Predictor - Client Bundle

How to run (double-click):

0) 0_install_packages.bat   [Run once on first install only]
1) 1_run_consolidado_downloader.bat
2) 2_run_data_processor.bat
3) 3_run_vin_trainer.bat
4) 4_run_sku_trainer.bat
5) 5_run_sku_predictor.bat  [Launch GUI]

Data lives in Source_Files (do not move it):
- Text_Processing_Rules.xlsx
- Maestro.xlsx
- Consolidado.json
- processed_consolidado.db

Logs will appear in logs/.

This folder is self-contained; do not copy files outside.
